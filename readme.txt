ASD-OTP-Sender
This OTP Sender is Good for sending messages if you have mail/SMS/WhatsApp to send. It uses VSC and Python(mandatory).

How does it work?

It generates a 4-digit code in VSC and then you have to send that and then the receiver(who is getting the OTP) will give you the Correct OTP. It will show 'Success!'if it is correct.
